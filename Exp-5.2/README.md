## ğŸ“– Theory

Route-based lazy loading is a performance optimization technique where components are loaded only when a specific route is accessed.

In React:

- `React.lazy()` is used to dynamically import components.
- `Suspense` is used to show a fallback UI while components load.
- `React Router DOM` enables navigation between different routes in a Single Page Application (SPA).

This technique reduces the initial bundle size and improves page load speed by splitting code into smaller chunks that are loaded only when required.
